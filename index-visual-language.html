Built from generative SVG, CSS 3D, and shader code — never Unsplash.

Five hero motifs rendered in code, ready to lift into the blog. Each motif is self-contained, animation-friendly, and respects the brief: real-feel infrastructure renders that earn their place on the page.

001 · Server Rack

Black-anodized 42U with blinking status LEDs.

Pure SVG: rack frame, rail mounts, individual U-slots, status LED matrix, ventilation patterns, and intentional cable spaghetti at the bottom — colored by VLAN.

Status LEDs animate at independent rates (steady, blink, fast-blink) to feel alive without being noisy. ~280 lines of inline SVG, scales to any size.

SVG · CSS keyframes · ~280 elements

002 · Silicon Die

H100-style die macro with bond wires and BGA pads.

Inline SVG of a chip package: photolithography grid (compute units), gold bond wires arcing from die edge to substrate, 18×18 BGA pad array, gold-plated I/O ring. Annotation lines call out die area, HBM stacks, NVLink lanes.

SVG · 580 paths · < 4kB gzipped

003 · Network Fabric

InfiniBand spine with animated packet pulses.

Dark-substrate SVG: 8-spine / 12-leaf Clos topology, animated dots traveling along link paths to simulate packet flow. Different link colors call out east-west traffic vs storage vs management plane.

Pulse animation uses animateMotion on a copy of each path — staggered, no JS needed.

SVG · <animateMotion> · zero JS
FABRIC · NDR 400G 12 LEAF · 8 SPINE
004 · Attention Map

Tokenized softmax over 16 heads, layer 24.

16×16 attention head matrix rendered as inline SVG with deterministic-but-organic intensity values. Tokens label both axes ("the", "model", "attends", "to"...). One head highlighted with a callout to show the brief diagonal attention pattern.

Generated procedurally — values derived from a hash of (query × key) so the pattern reads as natural attention, not random noise.

SVG · procedural fill · < 6kB
HEAD 8 · LAYER 24 · CTX 512 SOFTMAX
005 · Data Center Corridor

Hot-aisle perspective, raised floor, overhead busways.

Pure CSS 3D — no images. A perspective container holds a tilted floor with grid lines, four rack rows on either side using gradient strips for status LEDs, overhead pipe runs, and atmospheric haze at the vanishing point. Animates on its own; no JS.

Use as a fold backdrop or full-bleed hero. Layers blur and atmospheric overlay sell the depth.

CSS 3D · perspective + transform · zero deps
